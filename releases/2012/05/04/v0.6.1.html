Bitcoin version 0.6.1 released 4 May 2012

Bitcoin version 0.6.1 is now available for download at: http://sourceforge.net/projects/bitcoin/files/Bitcoin/bitcoin-0.6.1/

This is a bug-fix and code-cleanup release, with no major new features.

Please report bugs using the issue tracker at github: https://github.com/bitcoin/bitcoin/issues
